# HealthOps Mini / PainTrack Mini

Educational healthcare-style full-stack project for interview preparation.

## Stack

Backend:
- Python 3.12+
- FastAPI
- PostgreSQL
- SQLAlchemy 2.x
- Alembic
- Pydantic v2
- JWT authentication
- pytest

Frontend:
- React
- TypeScript
- Vite
- React Router
- Axios

Optional later:
- Django Integration Tracker mini-module
- Docker setup

## Project Structure

- backend-fastapi: FastAPI REST API
- frontend-react: React TypeScript frontend
- docs: project notes and learning plan
